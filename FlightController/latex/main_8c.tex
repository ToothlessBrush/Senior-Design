\doxysubsection{Detailed Description}
Main flight controller application for STM32\+F411-\/based quadcopter. 

This module implements the main control loop and state machine for a quadcopter flight controller. It coordinates \doxylink{structIMU}{IMU} sensor readings, \doxylink{structPID}{PID} control loops, motor control, and Lo\+Ra wireless communication.

The controller operates at 6660 Hz \doxylink{structIMU}{IMU} update rate and implements:\+
\begin{DoxyItemize}
\item Multi-\/state flight control (disarmed, arming, flying, manual, emergency stop)
\item \doxylink{structAttitude}{Attitude} stabilization using \doxylink{structPID}{PID} control on pitch, roll, and yaw
\item Velocity-\/based position corrections
\item Heartbeat-\/based safety monitoring with automatic emergency stop
\item Lo\+Ra-\/based command and telemetry communication
\item Manual motor control mode for testing and calibration
\end{DoxyItemize}

Compute motor speeds and send throttle commands to motors. 

Implements the motor mixing algorithm to combine base throttle, \doxylink{structPID}{PID} corrections, and per-\/motor biases into individual motor throttle values. The mixing accounts for the quadcopter X configuration and motor rotation directions.

Implements motor mixing for X-\/configuration quadcopter. Combines base throttle, \doxylink{structPID}{PID} corrections (pitch, roll, yaw), and individual motor biases to compute final throttle values for each motor.

Motor Mixing Algorithm (X configuration):\+
\begin{DoxyItemize}
\item Motor 1 (rear left, CW):\+ throttle -\/ pitch + yaw + bias1
\item Motor 2 (front right, CCW):\+ throttle + roll -\/ yaw + bias2
\item Motor 3 (rear right, CCW):\+ throttle -\/ roll -\/ yaw + bias3
\item Motor 4 (front left, CW):\+ throttle + pitch + yaw + bias4
\end{DoxyItemize}

Pitch correction:\+ Motor 4 (front) increases, Motor 1 (rear) decreases Roll correction:\+ Motor 2 (right) increases, Motor 3 (left) decreases Yaw correction:\+ CW motors (1,4) increase, CCW motors (2,3) decrease

Final throttle values are clamped to MIN\+\_\+\+THROTTLE-\/\+MAX\+\_\+\+THROTTLE range and converted to DSHOT protocol values (1-\/1001 range).

Main entry point and control loop for flight controller. 

Initializes all subsystems and runs the main state machine loop. The state machine coordinates system initialization, \doxylink{structIMU}{IMU} updates, \doxylink{structPID}{PID} control, motor control, and wireless communication.

\doxylink{main_8c_a5d74787dedbc4e11c1ab15bf487e61f8}{State} Flow:\+
\begin{DoxyItemize}
\item INIT:\+ Configure clocks, peripherals, sensors, and communication
\item DISARMED:\+ Wait for commands, allow configuration changes
\item CALIBRATE:\+ Perform \doxylink{structIMU}{IMU} gyro/\+accel bias calibration
\item ARMING/\+\+ARMING\+\_\+\+MANUAL:\+ Initialize motors and transition to active mode
\item FLYING:\+ Run high-\/speed control loop with \doxylink{structIMU}{IMU} (6660 Hz) and \doxylink{structPID}{PID} updates
\item MANUAL:\+ Direct motor control mode for testing
\item EMERGENCY\+\_\+\+STOP:\+ Safety state requiring manual reset
\end{DoxyItemize}

Safety Features:\+
\begin{DoxyItemize}
\item Heartbeat timeout monitoring (2 second timeout triggers emergency stop)
\item Max angle protection (commented out, configurable)
\item Automatic motor shutdown on communication loss
\end{DoxyItemize}

Never returns (infinite loop)
