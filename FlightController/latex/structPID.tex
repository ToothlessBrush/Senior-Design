\doxysubsection{Detailed Description}
Multi-\/axis \doxylink{structPID}{PID} context for cascaded attitude and velocity control. 

Contains five \doxylink{structPID}{PID} controllers:\+
\begin{DoxyItemize}
\item Three for attitude control (pitch, roll, yaw) -\/ inner loop
\item Two for velocity correction (X, Y axes) -\/ outer loop
\end{DoxyItemize}

The velocity correction PIDs adjust the attitude setpoints to counteract horizontal drift, implementing a simple position hold feature.
