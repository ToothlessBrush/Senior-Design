\doxysubsection{Detailed Description}
\doxylink{structPID}{PID} controller for quadcopter attitude and velocity control. 

Implements cascaded \doxylink{structPID}{PID} control system:\+
\begin{DoxyItemize}
\item Inner loop:\+ \doxylink{structAttitude}{Attitude} control (pitch, roll, yaw) using gyroscope feedback
\item Outer loop:\+ Velocity correction (position hold) adjusting attitude setpoints
\end{DoxyItemize}

The \doxylink{structPID}{PID} uses direct gyroscope rate feedback for the derivative term instead of differentiating error, providing cleaner derivative action and better noise immunity.

Features:\+
\begin{DoxyItemize}
\item Independent \doxylink{structPID}{PID} controllers for each axis
\item Integral windup protection with configurable limits
\item Output limiting for safety
\item Velocity-\/based position correction (prevents drift)
\item Telemetry tracking of individual P, I, D terms
\item Angle normalization for correct wraparound handling 
\end{DoxyItemize}

Calculate \doxylink{structPID}{PID} output for a single axis using rate feedback. 

Implements \doxylink{structPID}{PID} control with direct rate feedback for derivative term:\+
\begin{DoxyItemize}
\item P term:\+ proportional to angle error (setpoint -\/ measurement)
\item I term:\+ integral of angle error with anti-\/windup clamping
\item D term:\+ proportional to negative angular velocity (damping)
\end{DoxyItemize}

Angle error is normalized to ±π for correct wraparound handling. Final output is clamped to ±output\+\_\+limit. Individual terms are stored in \doxylink{structPIDController}{PIDController} for telemetry.

Update attitude \doxylink{structPID}{PID} controllers with current \doxylink{structIMU}{IMU} measurements. 

Reads current attitude from \doxylink{structIMU}{IMU} and calculates control outputs for pitch, roll, and yaw axes. Uses gyroscope rates for derivative term. Should be called at fixed rate (6660 Hz) after \doxylink{imu_8c_adc03ca49fd244054f4f7b7ca0afb3558}{IMU\+\_\+update()}.

Output is throttle correction in range ±output\+\_\+limit for each axis:\+
\begin{DoxyItemize}
\item Positive pitch output:\+ increase rear motors, decrease front motors (nose up)
\item Positive roll output:\+ increase right motors, decrease left motors (roll right)
\item Positive yaw output:\+ increase CW motors, decrease CCW motors (yaw right)
\end{DoxyItemize}

Update velocity correction PIDs to adjust attitude setpoints. 

Outer loop controller that adjusts pitch and roll setpoints based on measured horizontal velocity to prevent drift. Uses high-\/pass filtered acceleration and integrated velocity from \doxylink{structIMU}{IMU}.


\begin{DoxyItemize}
\item Velocity X correction adjusts pitch setpoint (forward/\+back position hold)
\item Velocity Y correction adjusts roll setpoint (left/\+right position hold)
\end{DoxyItemize}

If velocity\+\_\+correction\+\_\+enabled is false, simply copies base\+\_\+setpoints to setpoints. Should be called before \doxylink{pid_8h_a8d01657a359adeca98d06b804d3f48b5}{pid\+\_\+update()} in the control loop.
