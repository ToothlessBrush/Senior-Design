\doxysubsection{Detailed Description}
\doxylink{structIMU}{IMU} driver for LSM6\+DSL accelerometer/\+gyroscope sensor. 

Provides high-\/speed (6660 Hz) \doxylink{structIMU}{IMU} data acquisition with:\+
\begin{DoxyItemize}
\item 3-\/axis gyroscope (±500 dps range, converted to rad/\+s)
\item 3-\/axis accelerometer (±8g range)
\item Complementary filter for attitude estimation (roll, pitch, yaw)
\item Biquad low-\/pass filters for noise reduction (motor vibration rejection)
\item Bias calibration for gyro and accelerometer
\item Velocity estimation from high-\/pass filtered acceleration
\item Interrupt-\/driven data ready signal for precise timing
\end{DoxyItemize}

Hardware:\+ LSM6\+DSL via SPI1, data ready interrupt on PA0 (EXTI0) Update Rate:\+ 6660 Hz (150 μs period)

Coordinate System (NED -\/ North-\/\+East-\/\+Down):\+
\begin{DoxyItemize}
\item X:\+ Forward (nose direction)
\item Y:\+ Right (right wing direction)
\item Z:\+ Down
\item Roll:\+ Rotation about X (positive = right wing down)
\item Pitch:\+ Rotation about Y (positive = nose up)
\item Yaw:\+ Rotation about Z (positive = clockwise from above) 
\end{DoxyItemize}

Update \doxylink{structIMU}{IMU} with new sensor readings. 

Reads raw sensor data, applies scaling and calibration, runs low-\/pass filters, updates attitude estimate, and calculates velocity from high-\/pass filtered acceleration. Should be called at fixed 6660 Hz rate when \doxylink{imu_8h_ab572d62d9bf20cb3a4544900c89d5649}{imu\+\_\+data\+\_\+ready()} is true.

Processing pipeline:\+
\begin{DoxyEnumerate}
\item Read raw gyro and accel
\item Convert to physical units (rad/\+s, g)
\item Apply bias calibration
\item Apply biquad low-\/pass filters
\item Update attitude (complementary filter)
\item Estimate velocity (high-\/pass filter + integration)
\end{DoxyEnumerate}
