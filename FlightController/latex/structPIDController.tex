\doxysubsection{Detailed Description}
Single-\/axis \doxylink{structPID}{PID} controller with anti-\/windup. 

Implements \doxylink{structPID}{PID} control with direct rate feedback for derivative term. The D term uses measured angular velocity instead of error derivative, eliminating derivative kick and reducing noise sensitivity.
