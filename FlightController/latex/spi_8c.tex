Initialize SPI1 peripheral for LSM6\+DSL communication. 

Configures GPIO pins, SPI1 peripheral in master mode with CPOL=1, CPHA=1, and baud rate of APB2/\+64. Sets CS pin (PA4) to output high (inactive).

\begin{DoxyReturn}{Returns}
true if initialization successful, false if SPI enable failed 
\end{DoxyReturn}
