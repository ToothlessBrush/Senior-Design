Update \doxylink{structIMU}{IMU} with new sensor readings. 

Reads raw sensor data, applies scaling and calibration, runs low-\/pass filters, updates attitude estimate, and calculates velocity from high-\/pass filtered acceleration. Should be called at fixed 6660 Hz rate when \doxylink{imu_8h_ab572d62d9bf20cb3a4544900c89d5649}{imu\+\_\+data\+\_\+ready()} is true.

Processing pipeline:\+
\begin{DoxyEnumerate}
\item Read raw gyro and accel
\item Convert to physical units (rad/\+s, g)
\item Apply bias calibration
\item Apply biquad low-\/pass filters
\item Update attitude (complementary filter)
\item Estimate velocity (high-\/pass filter + integration)
\end{DoxyEnumerate}
