\doxysubsection{Detailed Description}
Sys\+Tick timer for millisecond timekeeping and delays. 

Provides Arduino-\/style \doxylink{systick_8h_acd8d13c328bebdf9eddea27ff9448c2d}{millis()} function and blocking \doxylink{systick_8h_ab7cce8122024d7ba47bf10f434956de4}{delay\+\_\+ms()} using ARM Cortex-\/\+M4 Sys\+Tick timer. Timer runs at system clock (100 MHz) with 1 ms interrupt period.

Features:\+
\begin{DoxyItemize}
\item 1 ms resolution counter
\item 32-\/bit millisecond counter (wraps after \texorpdfstring{$\sim$}{\string~}49.7 days)
\item Blocking millisecond delay function 
\end{DoxyItemize}
