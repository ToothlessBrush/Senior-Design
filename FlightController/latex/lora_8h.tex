\doxysubsection{Detailed Description}
Lo\+Ra wireless communication driver for REYAX RYLR998 modules. 

Provides AT command interface, data transmission, and reception handling for Lo\+Ra wireless communication. Supports both blocking and non-\/blocking send operations with automatic response parsing.

The driver communicates with REYAX RYLR998 Lo\+Ra modules via UART using AT commands. It handles module initialization, configuration, data transmission, and automatic response processing.

Usage:\+
\begin{DoxyEnumerate}
\item Call \doxylink{lora_8h_a379f3ab9c38a33c107d3c24f8e123d83}{lora\+\_\+init()} during system initialization
\item Call \doxylink{lora_8h_aacc88f32c7baeb924e13546e7147ba27}{lora\+\_\+service()} periodically in main loop to process responses
\item Use \doxylink{lora_8h_a8f742f0d10d4426bf4c1917e2ba18f08}{lora\+\_\+send\+\_\+string()} or \doxylink{lora_8h_a5b48a412a271c571523b60b6fd626814}{lora\+\_\+send\+\_\+string\+\_\+nb()} to transmit data
\item Check \doxylink{lora_8h_a8380b0ba75b8b603cb6ec0533399f43c}{lora\+\_\+data\+\_\+available()} and use \doxylink{lora_8h_a9261a11971cd94b0af847275df9fbb01}{lora\+\_\+get\+\_\+received\+\_\+data()} to receive 
\end{DoxyEnumerate}
