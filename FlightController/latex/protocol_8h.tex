\doxysubsection{Detailed Description}
Flight controller command and telemetry protocol definitions. 

Defines the wire protocol for communication between ground control station and flight controller. Supports both text-\/based command prefixes (FC:\+, ES) and hex-\/encoded binary payloads (ST:\+, SP:\+, HB:\+, etc.) for efficient transmission over Lo\+Ra links.

Protocol Format:\+
\begin{DoxyItemize}
\item Text commands:\+ "{}\+FC:\+\+START"{}, "{}\+FC:\+\+STOP"{}, "{}\+ES"{} (emergency stop)
\item Binary commands:\+ "{}\+PREFIX:\+\+HEXDATA"{} where HEXDATA is hex-\/encoded binary struct
\begin{DoxyItemize}
\item ST:\+ Set throttle (8 hex chars = 4 bytes float)
\item SP:\+ Set point (24 hex = 12 bytes, 3 floats)
\item HB:\+ Heartbeat (32 hex = 16 bytes, 4 floats)
\item TP:\+ Tune \doxylink{structPID}{PID} (42 hex = 21 bytes, 5 floats + 1 uint8\+\_\+t)
\item MB:\+ Motor bias (32 hex = 16 bytes, 4 floats)
\item CF:\+ Config sync (152 hex = 76 bytes, full configuration) 
\end{DoxyItemize}
\end{DoxyItemize}
