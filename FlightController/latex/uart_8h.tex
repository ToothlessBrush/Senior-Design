\doxysubsection{Detailed Description}
UART2 driver for Lo\+Ra module communication (USART2, PA2/\+\+PA3). 

Provides interrupt-\/driven RX with circular buffer and DMA-\/based TX for efficient serial communication at 115200 baud. Used for AT command interface with REYAX RYLR998 Lo\+Ra modules.

Hardware Configuration:\+
\begin{DoxyItemize}
\item USART2 on APB1 (50 MHz clock)
\item PA2:\+ TX (AF7)
\item PA3:\+ RX (AF7) with pull-\/up
\item RX:\+ Interrupt-\/driven with 256-\/byte circular buffer
\item TX:\+ DMA1 Stream 6, Channel 4 for efficient transmission
\end{DoxyItemize}

Features:\+
\begin{DoxyItemize}
\item 115200 baud rate
\item Interrupt-\/driven RX with overflow protection
\item DMA-\/based TX for non-\/blocking transmission
\item Circular RX buffer (power-\/of-\/2 size for fast masking) 
\end{DoxyItemize}
