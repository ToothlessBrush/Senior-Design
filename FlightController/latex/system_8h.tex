\doxysubsection{Detailed Description}
System initialization and onboard LED control. 

Provides system clock configuration and onboard LED control functions for STM32\+F411 microcontroller. LED is active-\/low on PC13 (STM32 "{}black pill"{} development board).

Clock Configuration:\+
\begin{DoxyItemize}
\item 25 MHz HSE (external crystal)
\item PLL configured for 100 MHz system clock
\item AHB:\+ 100 MHz
\item APB1:\+ 50 MHz (max for F411)
\item APB2:\+ 100 MHz 
\end{DoxyItemize}

void System\+Clock\+\_\+\+Config\+\_\+100\+MHz\+\_\+\+HSE (\begin{DoxyParamCaption}\item[{void}]{}{}\end{DoxyParamCaption})}



Configure system clock to 100 MHz using HSE and PLL. 

Configures STM32\+F411 to run at 100 MHz using 25 MHz external crystal:\+
\begin{DoxyItemize}
\item Enables HSE and waits for stabilization
\item Configures PLL:\+ (HSE /\+ 25) \texorpdfstring{$\ast$}{*} 200 /\+ 2 = (25 /\+ 25) \texorpdfstring{$\ast$}{*} 200 /\+ 2 = 100 MHz
\item Sets flash latency to 3 wait states (required for 100 MHz at 3.\+3V)
\item Enables instruction and data caches
\item Configures bus prescalers:\+ AHB /\+1, APB1 /\+2, APB2 /\+1
\item Switches system clock source to PLL
\end{DoxyItemize}

Clock Tree:\+
\begin{DoxyItemize}
\item SYSCLK:\+ 100 MHz
\item AHB (HCLK):\+ 100 MHz
\item APB1 (PCLK1):\+ 50 MHz (timers run at 100 MHz)
\item APB2 (PCLK2):\+ 100 MHz
\end{DoxyItemize}
