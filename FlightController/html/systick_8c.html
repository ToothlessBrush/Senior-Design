uint32_t millis ( void )

Get millisecond counter value.

Returns the number of milliseconds since systick_init() was called. Counter increments in SysTick_Handler ISR every 1 ms. Wraps to 0 after approximately 49.7 days (2^32 milliseconds).

void systick_init ( void )

Initialize SysTick timer for 1 ms interrupts.

Configures SysTick to generate interrupts every 1 ms at 100 MHz system clock. Enables SysTick interrupt and starts the timer. Must be called during system initialization before using millis() or delay_ms().
