Detailed Description

Multi-axis PID context for cascaded attitude and velocity control.

Contains five PID controllers:

  • Three for attitude control (pitch, roll, yaw) - inner loop
  • Two for velocity correction (X, Y axes) - outer loop

The velocity correction PIDs adjust the attitude setpoints to counteract horizontal drift, implementing a simple position hold feature.
